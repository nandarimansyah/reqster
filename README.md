# Reqster

Reqster is an open-source desktop API client for testing, debugging,
and exploring HTTP, REST, and JSON APIs.

## Features
- Send HTTP requests (GET, POST, PUT, DELETE, etc.)
- Environment & variable support
- Collections and folders
- Clean, fast, local-first desktop experience
- Fully open source

## Motivation
Reqster aims to be a lightweight, developer-friendly, desktop
alternative to Postman and Insomnia.
